# 🔧 설치 및 실행 가이드

## Windows 사용자용 간단 가이드

### 준비물
- Windows 10 이상
- 인터넷 연결 (처음 설치할 때만)

---

## 📥 방법 1: 실행 파일 사용 (가장 쉬움, 추천!)

### 1단계: 실행 파일 만들기

1. 이 폴더를 다운로드합니다.
2. `run.bat` 파일을 더블클릭합니다.
3. 처음 실행 시 필요한 프로그램들이 자동으로 설치됩니다.

### 2단계: .exe 파일 생성

1. `build_exe.py` 파일을 더블클릭하거나
2. 명령 프롬프트에서 다음 명령어 실행:
   ```
   python build_exe.py
   ```
3. ico 파일을 이용해 아이콘 파일을 만들기 위해서, `build.exe.py` 파일 안 command의 --icon 값에 ico 파일 경로를 추가합니다.

### 3단계: 프로그램 사용

1. `dist/사진PDF출력기` 폴더가 생성됩니다.
2. 이 폴더를 바탕화면이나 원하는 위치로 복사합니다.
3. `사진PDF출력기.exe`를 더블클릭하여 실행합니다.

**💡 팁**: `사진PDF출력기.exe` 파일을 오른쪽 클릭 → "바로가기 만들기"로 바탕화면에 바로가기를 만들면 더 편리합니다!

---

## 📥 방법 2: Python으로 직접 실행

### 1단계: Python 설치 확인

1. 명령 프롬프트(CMD)를 엽니다.
   - Windows 키를 누르고 "cmd" 입력
2. 다음 명령어를 입력:
   ```
   python --version
   ```
3. Python 버전이 표시되면 설치되어 있는 것입니다.
4. 설치되어 있지 않다면:
   - https://www.python.org/downloads/ 에서 다운로드
   - 설치 시 "Add Python to PATH" 체크!

### 2단계: 필요한 패키지 설치

명령 프롬프트에서 이 폴더로 이동한 후:

```bash
pip install -r requirements.txt
```

### 3단계: 프로그램 실행

```bash
python main.py
```

또는 `run.bat` 파일을 더블클릭합니다.

## 🎯 어르신께 설명하는 법

1. **실행 파일 방법 사용을 추천합니다.**
   - 한 번만 설치하면 계속 사용 가능
   - 바탕화면에 바로가기 만들어두기

2. **사용법 (3단계만 기억하면 됩니다)**
   ```
   1. 사진 선택하기 버튼 클릭
   2. 원하는 글귀 입력 (선택사항)
   3. "바로 인쇄하기" 버튼 클릭
   ```

3. **큰 화면에서 사용하기**
   - 모니터 화면 배율을 125% 또는 150%로 설정하면 더 잘 보입니다.
   - Windows 설정 → 시스템 → 디스플레이 → 배율

## ❓ 자주 묻는 질문

### Q: Python이 뭔가요?
A: 프로그래밍 언어입니다. 하지만 실행 파일(.exe)을 만들면 Python 없이도 사용 가능합니다!

### Q: 매번 설치해야 하나요?
A: 아니요! 한 번만 설치하면 됩니다. 실행 파일을 만들면 언제든지 클릭만 하면 됩니다.

### Q: 인터넷 없이도 사용 가능한가요?
A: 네! 한 번 설치 후에는 인터넷 없이도 사용 가능합니다.

### Q: 프린터가 없어도 되나요?
A: 네! PDF로 저장해서 나중에 인쇄하거나, USB에 담아서 다른 곳에서 인쇄할 수 있습니다.

## 🆘 문제가 생겼을 때

### "Python을 찾을 수 없습니다" 오류
→ Python 설치가 필요합니다. 또는 실행 파일(.exe)을 만들어 사용하세요.

### "모듈을 찾을 수 없습니다" 오류
→ `pip install -r requirements.txt` 명령어를 실행하세요.

### 프린터가 작동하지 않을 때
→ "PDF로 저장" 버튼을 사용한 후, PDF 파일을 직접 열어서 인쇄하세요.

### 한글이 깨질 때
→ 대부분 자동으로 해결됩니다. Windows의 "맑은 고딕" 폰트를 사용합니다.
