# 📸 사진 PDF 출력기

어르신도 쉽게 사용할 수 있는 사진-텍스트-PDF 변환 및 프린터 출력 프로그램입니다.

## ✨ 주요 기능

- **간단한 3단계 사용법**
  1. 사진 선택
  2. 글귀 입력 (선택사항)
  3. PDF 저장 또는 바로 인쇄

- **큰 버튼과 큰 글씨**로 어르신도 편하게 사용
- **이미지 미리보기**로 선택한 사진 확인
- **한글 완벽 지원**
- **A4 크기 PDF** 자동 생성
- **Windows 프린터 바로 출력**

## 🚀 설치 방법

### 방법 1: Python이 설치되어 있는 경우

1. 이 폴더를 다운로드합니다.
2. 명령 프롬프트(CMD)를 열고 이 폴더로 이동합니다.
3. 아래 명령어를 실행하여 필요한 패키지를 설치합니다:

```bash
pip install -r requirements.txt
```

4. 프로그램 실행:

```bash
python main.py
```

### 방법 2: 실행 파일(.exe) 만들기 (추천)

Python 설치가 없어도 실행 가능한 .exe 파일을 만들 수 있습니다.

1. PyInstaller 설치:

```bash
pip install pyinstaller
```

2. 실행 파일 생성:

```bash
python build_exe.py
```

3. `dist/사진PDF출력기` 폴더에 생성된 프로그램을 사용합니다.

## 📖 사용 방법

### 1단계: 사진 선택
- **"📁 사진 선택하기"** 버튼을 클릭합니다.
- 컴퓨터에서 원하는 사진을 선택합니다.
- 선택한 사진이 화면에 미리보기로 나타납니다.

### 2단계: 글귀 입력 (선택사항)
- 사진 아래에 넣고 싶은 글귀를 입력합니다.
- 글귀가 필요 없으면 비워두어도 됩니다.

### 3단계: PDF 저장 또는 인쇄
- **"💾 PDF로 저장"**: 원하는 위치에 PDF 파일로 저장합니다.
- **"🖨️ 바로 인쇄하기"**: 기본 프린터로 바로 인쇄합니다.

## 💡 팁

- **지원되는 이미지 형식**: JPG, JPEG, PNG, BMP, GIF
- **PDF 크기**: A4 용지 크기로 자동 조정됩니다.
- **이미지 품질**: 원본 이미지의 비율을 유지하면서 A4에 맞게 조정됩니다.
- **여러 줄 입력**: 글귀를 여러 줄로 입력할 수 있습니다.

## 🔧 시스템 요구사항

- **운영체제**: Windows 10 이상 (프린터 기능)
- **Python**: 3.8 이상 (소스코드 실행 시)
- **메모리**: 최소 2GB RAM
- **디스크**: 100MB 이상 여유 공간

## ⚠️ 문제 해결

### 한글이 깨져 보이는 경우
- Windows의 "맑은 고딕" 폰트가 설치되어 있어야 합니다.
- 대부분의 Windows 시스템에는 기본 설치되어 있습니다.

### 프린터가 작동하지 않는 경우
- Windows에서 기본 프린터가 설정되어 있는지 확인하세요.
- 프린터 드라이버가 최신 버전인지 확인하세요.
- PDF로 저장한 후 수동으로 인쇄하는 방법도 있습니다.

### 이미지가 표시되지 않는 경우
- 지원되는 이미지 형식인지 확인하세요.
- 이미지 파일이 손상되지 않았는지 확인하세요.
